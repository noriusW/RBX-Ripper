**RBX Ripper Pro** is an ultra-fast and user-friendly tool for extracting resources from Roblox place files (`.rbxlx`). Built with Rust to ensure maximum performance and safety.

### ‚ú® Features

- ‚ö° **Blazing Speed**: Powered by Rust and `roxmltree`, it handles massive files in seconds.
- üìÇ **Full Hierarchy**: Recreates the Roblox project structure using native folders.
- üìú **Script Export**: All `Script`, `LocalScript`, and `ModuleScript` objects are extracted as `.lua` files.
- ‚öôÔ∏è **Property Preservation**: All object properties are saved into `properties.json`.
- üåç **Multilingual**: Automatic RU/EN detection with manual toggle support.
- üñ±Ô∏è **Drag-and-Drop**: Simply drop your file into the application window.
- üìä **Progress Visuals**: Real-time progress bar with object counters.

```bash

cd rbx_ripper

```
